# Manual / External Installs

## Google Chrome
- Download .deb from official Google site
- sudo dpkg -i google-chrome-stable_current_amd64.deb

## MongoDB Compass
- Download .deb from mongodb.com
- Used only as a client (databases run in Docker)

## Arduino IDE
- Prefer native Linux tarball (not Snap)
- Extract to ~/Arduino
- Add binary to PATH if needed

## Cursor
- Installed via official .deb
- AI-assisted editor (optional)

## OnlyOffice
- Installed via .deb when needed

## Xtreme Download Manager (XDM)
- Installed manually
- Stores data in ~/.xdman (not tracked)

## VS Code Extensions
- Reinstalled manually or via settings sync
